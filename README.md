  #### 위 링크의 분석내용
  <b>
  위 링크에서 [std_id=뒤의 부분은 학생(자신)의 id], [sem_id=뒤의 부분은 현재 학기의 id], [top_cors_id=뒤의 부분은 현재 확인 불가], [cid=뒤의 부분은 chapter id 즉 현재 챕터의 고유 id], [g_seq=뒤의 부분은 현재 숙제의 고유 id]</b>
